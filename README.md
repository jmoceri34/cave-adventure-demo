# cave-adventure-demo

> This demo shows how you can use a camera to follow a character using overrides to control the viewport.

* [Overview](#overview)

<a name="overview"></a>
## Overview

Controls:

* Left : a | left-arrow
* Right: d | right-arrow
* Jump : spacebar
* Reset: r
